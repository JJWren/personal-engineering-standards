# Assets

Backgroundless logo derived from the source artwork (gray background removed, alpha transparency).

| File | Description |
| --- | --- |
| `logo.svg` | Scalable SVG with the transparent logo embedded (no background). |
| `logo.png` | Transparent PNG master (squared, centered). |
| `logo.ico` | Multi-resolution icon containing 16×16, 32×32, and 48×48. |
| `logo-16.ico` | Single 16×16 icon. |
| `logo-32.ico` | Single 32×32 icon. |
| `logo-48.ico` | Single 48×48 icon. |

All files use full alpha transparency in place of the original gray background.
